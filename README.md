 🌐 Collabify

 Empowering Research & SDG-Driven Innovation

Collabify is a networking platform connecting **researchers, students, and innovators** to collaborate on impactful projects aligned with **SDG Goal 9 (Industry, Innovation & Infrastructure)**.

---

 ✨ Key Features

* 🔎 **Real-Time Collaborator Search** – Find partners by skills, field, or institution
* 📄 **Live Profile Preview** – Generate a professional research profile
* 💡 **SDG Idea Board** – Share and discover project ideas
* 🔔 **Smart Notifications** – Clean toast alerts
* 📱 **Responsive Design** – Works on all devices

---

🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** Node.js, JSON-Server
* **Database:** `db.json`

---

🚀 Run Locally

```bash
cd backend
node seed.js
node seed_ideas.js
node server.js
```

📍 Backend: http://localhost:3000

👉 Open `index.html` with **Live Server** (recommended)
